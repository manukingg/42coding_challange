# Filter Prime Numbers

Difficulty: Medium

## Problem

You are given an integer array `nums`.

Return a new array containing only the prime numbers from `nums`, while keeping
the same order as in the input array.

If no prime numbers are found, return an empty array.

## Function Signature

```c
int* filterPrimes(int* nums, int numsSize, int* returnSize);
```

## Examples

```text
Input: nums = [1, 2, 3, 4, 5, 6]
Output: [2, 3, 5]
```

```text
Input: nums = [8, 9, 10, 12]
Output: []
```

```text
Input: nums = [11, 4, 13, 15, 17]
Output: [11, 13, 17]
```

## Notes

- Prime numbers are integers greater than `1` with exactly two positive divisors:
  `1` and the number itself.
- Negative numbers, `0`, and `1` are not prime.
- Keep the output order the same as the input order.
- Follow the standard LeetCode C pattern: return a dynamically allocated array
  and set `*returnSize` to the number of prime values in the result.
